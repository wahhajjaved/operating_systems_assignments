Description

In this lab, you will be implementing a round robin scheduler with 5 priorities using using multiple queues. It is possible to use a heap, but you are encouraged to use your list library from assignment 1 for the queues.

Inside set_state() add processes that are being set to RUNNABLE to the queue based on their priority. The priority of a process is inside its process table entry. Reimplement next_proc() to return the highest priority runnable process. Treat priority like POSIX does, i.e. a process with a lower priority number has "highest priority".
